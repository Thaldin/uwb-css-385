Key Controls
	W/S, Up/Down = Speed up/down
	A/D, Left/Right = Rotate left/right
	Space = fire
	P = Stop arrow
	M = Toggle between keyboard or mouse movement

WayPoint Controls
	H = Toggle waypoint visibility
	J = Toggle waypoint sequencing
	R = Reset waypoints

Game Controls
	Q = Quit to title
